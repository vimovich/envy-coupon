# Envy Nails by Face — подарочный купон (сайт)

Простой статический сайт с переворачивающимся купоном:
- **Лицевая сторона** — ваша готовая картинка (`assets/front.png`).
- **Обратная сторона** — собрана из кода (не картинка), фон сделан "переливающимся" (анимированный холографический градиент), а плашки с текстом — эффект жидкого стекла (blur + блики).
- На телефоне в портретном положении сайт **автоматически разворачивается в альбомную ориентацию** на весь экран.
- **Свайп или тап** по купону переворачивает его на другую сторону (работает и мышкой на компьютере — для проверки).

## Структура файлов

```
index.html      — разметка
style.css       — все стили, анимации, эффект стекла, поворот на мобильном
script.js       — переворот по тапу/свайпу
assets/front.png — ваша картинка лицевой стороны
```

## Как проверить локально

Просто открыть `index.html` в браузере (двойной клик), либо через локальный сервер:

```bash
npx serve .
```

## Как опубликовать через GitHub Pages

1. Создайте новый репозиторий на GitHub и загрузите туда все файлы этой папки (включая `assets/front.png`).
2. Зайдите в **Settings → Pages**.
3. В **Source** выберите ветку `main` и папку `/root`.
4. Сохраните — через минуту сайт будет доступен по адресу вида
   `https://<ваш-логин>.github.io/<название-репозитория>/`.

## Как опубликовать через Vercel

1. Загрузите репозиторий на GitHub (как выше).
2. Зайдите на vercel.com → **Add New → Project**.
3. Выберите этот репозиторий. Никаких настроек сборки не нужно — это чистый статический сайт (Framework Preset: **Other**).
4. Нажмите **Deploy**.

## Что можно поправить под себя

- **Текст купона** — в `index.html`, внутри блока `<!-- BACK -->`.
- **Цвета** — в начале `style.css`, переменные `--pink`, `--maroon`, `--maroon-deep`.
- **Скорость перелива фона** — `animation: holoMove 14s ...` в `.holo-bg` (уменьшите число — переливание будет быстрее).
- **Сила эффекта стекла** — свойство `backdrop-filter: blur(14px)` в `.glass-row` (больше значение — сильнее размытие).
- **Ссылки соцсетей** — в футере обратной стороны (`href="https://instagram.com/..."`, `href="https://t.me/..."`, `href="tel:..."`).
