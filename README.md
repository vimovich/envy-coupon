# Envy Nails by Face — подарочный купон (сайт)

**Теперь всего 3 файла** — картинка лицевой стороны зашита прямо в `index.html` (в base64), поэтому больше никакой папки `assets` и никаких сломанных путей к картинке при загрузке на GitHub.

```
index.html   — разметка + картинка лицевой стороны внутри
style.css    — все стили, анимации, эффект стекла, поворот на мобильном
script.js    — переворот по тапу/свайпу
```

## Что было исправлено в этой версии

1. **Купон был маленький, с кучей пустого места вокруг** — причина: при принудительном повороте экрана на телефоне ширина/высота купона считались от `vw`/`vh` реального (портретного) экрана, а не от того, как это выглядит после поворота. Теперь на мобильном эти оси намеренно поменяны местами — купон занимает почти весь экран.
2. **Иконки-преимущества и подписи разъезжались в 2 строки** — переверстала в один ряд: иконка + подпись в одну строку, без переносов.
3. **Контакты (Instagram/Telegram/телефон) пропадали** — это было следствие пункта 1: из-за того что купон рендерился маленьким, текст (с фиксированным минимальным размером) не помещался по высоте и обрезался. Теперь всё меряется в единицах, привязанных к реальному размеру самого купона (container query units), а не ко всему экрану — при любом размере экрана всё масштабируется вместе и ничего не обрезается.
4. **Заголовок «Подарочный купон»** — увеличила и закрепила в одну строку (`white-space: nowrap`).
5. **Картинка на лицевой стороне не грузилась после загрузки на GitHub** — потому что не хватало папки `assets`. Теперь картинка встроена прямо в код, отдельно её загружать не нужно вообще.

## Как проверить локально

Просто открыть `index.html` в браузере (двойной клик), либо через локальный сервер:

```bash
npx serve .
```

## Как опубликовать через GitHub Pages

1. Создайте репозиторий на GitHub и загрузите туда все 3 файла (`index.html`, `style.css`, `script.js`) — просто перетащите их в окно загрузки, никаких папок.
2. Settings → Pages → Source: ветка `main`, папка `/root` → Save.
3. Через минуту сайт будет доступен по адресу вида `https://<логин>.github.io/<репозиторий>/`.

## Как опубликовать через Vercel

1. Загрузите те же 3 файла в репозиторий на GitHub.
2. vercel.com → Add New → Project → выберите репозиторий → Framework Preset: **Other** → Deploy.

## Что можно поправить под себя

- **Текст купона** — в `index.html`, внутри блока `<!-- BACK -->`.
- **Цвета** — переменные `--pink`, `--maroon`, `--maroon-deep` в начале `style.css`.
- **Скорость перелива фона** — `animation: holoMove 14s ...` в `.holo-bg`.
- **Сила эффекта стекла** — `backdrop-filter: blur(14px)` в `.glass-row`.
- **Ссылки соцсетей** — в футере обратной стороны (`href="https://instagram.com/..."` и т.д.).
- **Если захотите заменить картинку лицевой стороны** — самый простой способ: попросите меня пересобрать файл с новой картинкой (так она снова встроится в код без папок и ссылок).
